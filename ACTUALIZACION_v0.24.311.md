# EncisoMath LMS v0.24.311

## Cambios

- Controles Cuadrícula/Lista compactos, a la derecha y con el color del tema en Clases, Actividades y Quizzes.
- El cambio de vista ya no reconstruye la pestaña completa; solo cambia el contenedor de tarjetas.
- Visor PDF rediseñado para aprovechar el ancho en celular y la altura en escritorio.
- Animación de hoja suavizada con doble lienzo.
- Corrección del mensaje permanente “Preparando cuaderno”.
- Zoom con botones, rueda Ctrl/Cmd y pellizco táctil; arrastre para desplazarse cuando hay zoom.
- Papelera para eliminar una clase. Si está compartida, permite quitarla solo del curso actual o eliminarla de todos.

No requiere migración SQL adicional.
